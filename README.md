## Overview

This repository contains the work completed for the SIEM & Log Analysis Foundations assignment as part of the Sohail Smart Solutions Summer Training Program 2026.

The objective of this exercise was to analyze security logs, identify suspicious events, and document potential security threats that may require investigation by a Security Operations Center (SOC) team.

Through this project, I reviewed sample security logs, identified indicators of suspicious activity, documented key findings, and recommended response actions that a SOC analyst may take during an investigation.

## Repository Structure

### sample-logs/

* Sample authentication and security log files used during analysis.

### notes/

* Investigation notes, observations, and event analysis.

### screenshots/

* Screenshots showing log analysis evidence and findings.

### report/

* Final SIEM and log analysis report.

## Key Security Events Investigated

1. Multiple failed login attempts from the same IP address.
2. Successful login after repeated authentication failures.
3. Unauthorized privilege escalation activity.
4. Network reconnaissance and port scanning behavior.
